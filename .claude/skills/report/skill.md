# 個別レポートスキル

個別銘柄の財務分析レポートを生成するスキル。

## 使い方

ユーザーが「この銘柄を分析して」「レポートを出して」「財務分析して」等と指示した場合にこのスキルを使用。

## 実行方法

```bash
cd /home/user/screening-test
uv run screening-test report {ticker}
```

## パラメータ

- `ticker`: ティッカーシンボル（例: `7203.T`, `AAPL`, `0700.HK`）

## レポート内容

- 企業基本情報（社名、セクター、時価総額）
- バリュエーション指標（PER、PBR）
- 収益性指標（配当利回り、ROE、売上成長率）
- 株価情報（現在値、52週高値/安値）
- バリュースコア（100点満点）と投資判定

## 使用例

- 「トヨタを分析して」→ `report 7203.T`
- 「Appleのレポート」→ `report AAPL`
- 「テンセントの財務分析」→ `report 0700.HK`
